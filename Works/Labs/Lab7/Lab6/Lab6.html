What exactly is a favicon?

A favicon is, as google defines it:

While they aren't necessarily the most important thing for a website. They help users to recognize the owner(s) of the website.

How are Favicons created?

Favicons are created by first using some sort of editing tool whether it be GIMP, Photoshop, etc. to create the image.
Once your image is ready you can add it to your website using HTML. Like I did!

How did I create the favicon for this site?

This is the favicon I created for this website! I created it using a pixelation tool that allows me to use Binary/Hexadecimal to create an image. The image I created was fairly simplistic in nature as it's only 1 bit per pixel. The actual size of the image is 32x32. Just for reference, I'll post it here: Pretty small, right?
